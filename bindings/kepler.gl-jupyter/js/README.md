This is a simple jupyter widget for kepler.gl, an advanced geospatial visualization tool, to render large-scale interactive maps.

Package Install
---------------

**Prerequisites**
- [node](http://nodejs.org/)
- [yarn]((https://yarnpkg.com/en/docs/install#mac-stable)

**More links**
